üìñ Apa itu Stack?

Stack adalah struktur data linear yang mengikuti prinsip LIFO (Last-In-First-Out) ‚Äî elemen yang terakhir dimasukkan akan menjadi elemen pertama yang dikeluarkan.

üéñÔ∏è Konteks Militer

Dalam operasi militer, konsep stack sangat relevan:

üíª Implementasi Stack dengan Array

Menggunakan array dengan ukuran tetap (fixed size)

  • topIndex = -1 menandakan stack kosong
  • Push: ++topIndex, lalu simpan data
  • Pop: Ambil data, lalu --topIndex

Placeholder: images/p05-03-array-memory-layout.png

üíª Class StackArray - Struktur


class StackArray {
private:
    static const int MAX_SIZE = 100;
    int data[MAX_SIZE];
    int topIndex;  // -1 jika kosong
    
public:
    StackArray() : topIndex(-1) {}
    
    bool isEmpty() { return topIndex == -1; }
    bool isFull()  { return topIndex == MAX_SIZE - 1; }
};

üíª Operasi Push


void push(int value) {
    if (isFull()) {
        cout << "Error: Stack Overflow!" << endl;
        return;
    }
    data[++topIndex] = value;
}

‚ö†Ô∏è Stack Overflow: Terjadi ketika push ke stack yang sudah penuh

üíª Operasi Pop & Peek


int pop() {
    if (isEmpty()) {
        cout << "Error: Stack Underflow!" << endl;
        return -1;
    }
    return data[topIndex--];
}

int peek() {
    if (isEmpty()) {
        cout << "Error: Stack kosong!" << endl;
        return -1;
    }
    return data[topIndex];
}

üíª Operasi Push (Linked List)


void push(int value) {
    Node* newNode = new Node(value);
    newNode->next = topNode;  // Arahkan ke top lama
    topNode = newNode;        // Update top
    count++;
}

‚úÖ Kompleksitas: O(1) - selalu di head

üíª Operasi Pop (Linked List)


int pop() {
    if (isEmpty()) {
        cout << "Stack Underflow!" << endl;
        return -1;
    }
    
    Node* temp = topNode;
    int value = temp->data;
    topNode = topNode->next;  // Update top
    delete temp;              // Hapus node lama
    count--;
    return value;
}

üìñ Aplikasi: Balanced Parentheses

Memeriksa apakah tanda kurung () [] {} dalam string sudah seimbang.

‚úì Balanced: "([]){}", "{[()]}"

‚úó Not Balanced: "([)]", "((("

üìñ Algoritma Balanced Parentheses

  1. Untuk setiap karakter:
  2. Jika kurung buka ‚Üí PUSH
  3. Jika kurung tutup ‚Üí POP & CHECK
  4. Di akhir: stack harus kosong

üíª Implementasi isBalanced()


bool isBalanced(string expr) {
    stack<char> s;
    for (char c : expr) {
        if (c == '(' || c == '[' || c == '{')
            s.push(c);
        else if (c == ')' || c == ']' || c == '}') {
            if (s.empty()) return false;
            char top = s.top();
            s.pop();
            if (!isMatchingPair(top, c))
                return false;
        }
    }
    return s.empty();
}

üí° Mengapa Postfix?

  • ‚úÖ Tidak memerlukan tanda kurung
  • ‚úÖ Tidak perlu aturan prioritas saat evaluasi
  • ‚úÖ Mudah dievaluasi dengan stack
  • ‚úÖ Digunakan di kalkulator dan compiler

üìä Prioritas Operator

Operator Prioritas Associativity
^ 3 (tertinggi) Right to Left
* / 2 Left to Right
+ - 1 (terendah) Left to Right

üìñ Algoritma Infix ke Postfix

Shunting-Yard Algorithm:

  1. Operand ‚Üí langsung ke output
  2. ( ‚Üí push ke stack
  3. ) ‚Üí pop sampai (
  4. Operator ‚Üí pop operator dengan prioritas ‚â•, lalu push
  5. Akhir ‚Üí pop semua sisa operator

üìñ Evaluasi Ekspresi Postfix

Algoritma:

  1. Operand ‚Üí PUSH ke stack
  2. Operator ‚Üí POP dua operand, hitung, PUSH hasil
  3. Akhir ‚Üí nilai di top adalah hasil

üíª Implementasi evaluatePostfix()


int evaluatePostfix(string postfix) {
    stack<int> operandStack;
    for (char token : postfix) {
        if (isdigit(token))
            operandStack.push(token - '0');
        else {
            int op2 = operandStack.top(); operandStack.pop();
            int op1 = operandStack.top(); operandStack.pop();
            int result;
            switch (token) {
                case '+': result = op1 + op2; break;
                case '-': result = op1 - op2; break;
                case '*': result = op1 * op2; break;
                case '/': result = op1 / op2; break;
            }
            operandStack.push(result);
        }
    }
    return operandStack.top();
}
